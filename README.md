## Power BI Data Visualization Project

Final project created as part of a Data Visualization and Business Intelligence course.

### Project overview
The goal of the project was to analyze a dataset and present insights
using interactive dashboards and visual elements.

### Key features
- Interactive charts and visualizations
- Use of filters and slicers
- Clear presentation of key metrics
- Business-oriented data interpretation

### Tools
- Power BI
- Data modeling
- Data visualization principles
